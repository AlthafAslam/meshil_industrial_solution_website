# Meshil Industrial Solutions Website

A high-performance, server-side rendered (SSR) website for Meshil Industrial Solutions, built with Next.js.

## Prerequisites

- Node.js 18.17 or later.
- npm, yarn, or pnpm.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

2.  **Run the Development Server**:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `app/`: Application source code (App Router).
    -   `layout.js`: Global root layout.
    -   `page.js`: Homepage.
    -   `globals.css`: Global styles.
    -   `components/`: Reusable components (Navbar, Footer).
-   `public/`: Static assets.

## Features

-   **Server-Side Rendering (SSR)**: Optimized for SEO and performance.
-   **Responsive Design**: Mobile-friendly layout.
-   **CSS Modules**: Modular and maintainable styling.
-   **SEO Optimized**: Semantic HTML and metadata configuration.
